For users of Connections Business Directory. This adds a Special Offers section allowing you to manage and schedule offers using start and end dates.

== Description ==

This simple plugin will add a visual text editor and deal scheduler to the Special Offers custom field. It converts content to html.  It lets you schedule multiple deals. For example, create monthly deals, year long deals, holiday deals and more.

= Key Features =

* Add multiple special offers to the Connections Business Directory Special Offers area.
* Offers are automatically made visible on the front-end starting on the Visible Date or Start Date you enter for each special offer.
* Offers are hidden after the end date that you enter for each special offer.
* Each offer is reformatted for better display on the front-end and can be customized using CSS classes.

This is an add on plugin for the Connections Business Directory Plugin please be sure to install and active it before adding this plugin.

What does this plugin do? It adds the ability to add offers to a listing by incorporating a visual text editor and deal scheduler to the Special Offers custom field. It converts content to html code so that you don't have to.

In addition to the text editor, you can create and schedule multiple deals. For example, create monthly deals, year long deals, holiday deals and more.


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/connections-business-directory-offers` directory or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. No configuration required.


Must have Connections Business Directory installed.


== Frequently Asked Questions ==

= How do I create a link to the voucher that I uploaded? =

Add a short code to the Offer Description in the Special Offers editor.  Use [VOUCHER link name], where link name is the text that will be displayed for the hyperlink to the voucher image.

= Do I have to update my existing offers when I change the date format? =

Unfortunately, the answer is yes.  Existing offers will continue to display using the previous format until you update the offer in the Offer Editor.

= Which custom field does the text editor appear in Connections Business Directory =

You need to create a custom field called Offers or Special Offers. The Edit Offers button is added next to the Delete button for the Offers field. The visual text editor adds html to the Connections Business Directory field for you. No knowledge of coding required to design your special offer description and format multiple offers at one time. You can customize with CSS.


= What are the class names for custom CSS? =


Title: tsl_connections_offers_fe_title
Starts Label: tsl_connections_offers_fe_starts_label
Start Date: tsl_connections_offers_fe_starts_data
Ends label: tsl_connections_offers_fe_ends_title
Ends data: tsl_connections_offers_fe_ends_data
Content: tsl_connections_offers_fe_content
Line break at end of each item: tsl_connections_offers_fe_row_sep
Horizontal line at the end of each item: tsl_connections_offers_fe_hr
Spacing at the end of the special offers section: tsl_connections_offers_fe_end_of_data

= Is it visible for external listing owners? =

It is visible to both listing owners and website admin.
